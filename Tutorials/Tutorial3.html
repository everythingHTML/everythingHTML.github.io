Tutorial 3: Text and headings

Of course, the actual text you can read on a website is pretty much the most important part of creating a website in the first place. It is the essential compound of any available online resource, but writing a text for a website is not as easy as when you write an essay in, for example, Microsoft Word as there are differnt types of tags for different types of text in HTML.

  1. The Heading

    2. The heading is the beginning of every website. It tells the reader what the content is about and also should aim to wake up the reader's attention.
    In HTML, we create headings by using the <h1> - <h9> tag. The number within the tag denotes the priority of the tag. <h1> resembles the biggest and also the primary heading and every following number becomes a smaller heading.

    Example:

    <h1>Heading 1</h1>

    Heading 1
    <h2>Heading 2</h2>

    Heading 2
    <h3>Heading 3</h3>

    Heading 3
    <h4>Heading 4</h4>

    Heading 4
    ...
    <h9>Heading 9</h9> Heading 9

  2. The Paragraph (aka the main text)

    3. As you might have already guessed, the heading is only the beginning of any website, the real content is the actual text, the paragraphs you really want to read. What you read right here is also this kind of text. Since the main text should always be written in paragraphs, the tag for it is simply called <p>.

    Example:

    <p>This is your paragraph here. You can write anything you want here and how long.<p/>

    Do note though, that the tag will not recognize any line breaks you do in the code. It will just recognize them as another empty space. In order to actually create a line break you have to use the tag <br/> (do note though, that this tag does not need a closing tag as this is already within the opening tag)

    Example:

    <p>This is a paragraph <br/> with a line break.</p>

    This is a paragraph
    with a line break.

  3. Types of Text

    4. Now that we know how we can create a heading and how to write a paragraph or a text in general, we should also talk about the different text styles. In HTML you can do many things with the text and the letters.

    Bold letters

    In order to create bold letters,we have to use the tag <b> inside the <p> tag.

    Example:

    <p><b>This paragraph is bold</b></p>

    This paragraph is bold

    Italic Letters

    In order to create italic letters,we have to use the tag <i> inside the <p> tag.

    Example:

    <p><i>This paragraph is italic</i></p>

    This paragraph is italic

    Underlined Letters

    In order to create underlined letters,we have to use the tag <u> inside the <p> tag. Do note though, that you should avoid using this tag as underlined words can easily be confused with a link.

    Example:

    <p><i>This paragraph is underlined</i></p>

    This paragraph is underlined

    Other text styles and use cases.

    Even if all of the examples mentioned above were always affecting the entire paragraph, you can also just affect certain parts of a paragraph, or just a word. There is also the possability of mixing as many text styles as you want, so, as an example you can make a word/text/paragraph bold and italic at the same time.

    Example:

    <p>Only <b>this</b> and those<b><i>two words</i></b> are affected by the text style.</p>

    Only this and those two words are affected by the text style.
