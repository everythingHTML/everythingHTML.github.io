The basic page design:

Every HTML document follows a certain structure and has to include some tags. This will also be the point where we will be starting to learn the first couple of tags, starting with

<!DOCTYPE html>

This tag defines the type of the file, in our case it is a HTML document, which is why we put the html in there. This tag has to always appear in the very first line of the HTML document, since that is where the browser will start reading your code. It has to be on every .html file you have and has to always appear on the top of it.

The next tag, every file will need is

<html>

Please note, that this tag is in no way related to <!DOCTYPE> since this one shows the browser where your html code will actually begin and end. It should only appear once in your file with the opening tag appearing right beneath <!DOCTYPE> and the closing tag should be the very last tag of the entire code.
