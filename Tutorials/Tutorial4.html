Tutorial 4: Adding images and listing things

Adding an image to your website will make it a lot nicer to look at and easier to navigate. To do this we need to use the image tag which looks like this:

&lt;img /&gt;

Note that this tag does not have a closing tag, instead it ends with a forward slash. You also need to add the source (src) of where the image is saved, with quotation marks at the start and end of the image name.

&lt;img src="[image name]" /&gt;

Adding an Image from the internet

To add an image from the internet, find it on your web browser and click on “view image”. Then copy the URL address and paste it as the source

&lt;img src=”[URL]” /&gt;

Adding an image from your computer

If the image is stored on your computer it is recommended that you create folder inside your website folder and call it “Images”, this will make it easier to be organised. After having made that file, add your desired image to the folder. In the img tag add [ ../ ], this tells the computer that the images is inside the website folder. Also add the type of image at the end of its name, usually .jpg or .png

&lt;img src=”../images/[name.jpg]” / &gt;

Listing Things

To create a list, you can either use an unordered list (bullet points) or an ordered list (numbers). For each point or number use the <li> tag, it has the closing tag </li> with your text between. An unordered list uses the <ul> tag and has the closing tag </ul>. An ordered list has the tag <ol> and the closing tag </ol>.

Example: Unordered list (Bullet point)
<ul>
  <li> [First bullet point] </li>
  <li> [Second bullet point] </li>
</ul>
Ordered List (Numbered)
<ol>
  <li> [First numbered point] </li>
  <li> [Second numbered point] </li>
<ol>
